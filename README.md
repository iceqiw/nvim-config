# Neovim

## Get healthy

Open `nvim` and enter the following:

```
:checkhealth
```

Next we need to install python support (node is optional)

- Neovim python support

  ```
  pip install pynvim
  ```
- Neovim node support

  ```
  npm i -g neovim
  ```

## Pre install

choco install ripgrepy

choco install fd
